# Project Team Manager

A small console application for managing a software house's project team. Demonstrates Java OOP concepts: inheritance, polymorphism, method overriding, `instanceof`, `final` methods, and `Object` class methods.

## Classes

- **Employee** – parent class with shared fields (ID, name, base salary)
- **Developer** – employee with main language and repository count
- **Tester** – employee with automation flag and scenario count
- **ProjectTeam** – manages the list of employees and computes total cost
- **Main** – demonstrates the full usage scenario
